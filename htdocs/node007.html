Logical Structure

Here is a snippet named logic.t which was originally written as a single line: 
if ((!($from || $to || $subject || $size || ($fieldname&&$fieldval)) || !$ruletype) && !$dest) {
        next;
}




The command perltidy logic.t produces a file
named logic.t.tdy with this result:



if (
    (
           !( $from || $to || $subject || $size || ( $fieldname && $fieldval ) )
        || !$ruletype
    )
    && !$dest
  )
{
    next;
}
